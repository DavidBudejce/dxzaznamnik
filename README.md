# DX záznamník
Hledač lokátorů a záznamník rádiového spojení, vhodný třeba pro navazování spojení s PMR.

Aplikace která spojuje vyhledávání lokátoru a záznamník spojení. Je třeba mít zapnutou GPS v mobilu (chvíli počkat až se chytí), pak odsouhlasit svou pozici vysílání podle mapy, ukáže se můj aktuální lokátor a pak mohu tam dolu zadávat spojení co navážu. Pamatuje si to spojení i když tu stránku zavřete. Dá se podobně pokračovat z jiného lokátoru a následující dny a dá se pak pěkně uložit přehled spojení třeba za celý měsíc. Dá se tam udělat export do CSV a to pak použít při zadávání do deníku na cbpmr. Vypisuje to vzdálenost spojení a které bylo nejdelší.
